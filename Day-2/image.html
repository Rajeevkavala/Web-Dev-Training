Images

Images can be embedded in HTML using the <img> tag. The src attribute specifies the path to the image file, and the alt attribute provides alternative text for the image.

Image Attributes

Here are some common attributes used with the <img> tag:

  • src: Specifies the path to the image file.
  • alt: Provides alternative text for the image, which is displayed if the image cannot be loaded.
  • width: Sets the width of the image in pixels.
  • height: Sets the height of the image in pixels.
